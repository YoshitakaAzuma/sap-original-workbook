## 一問一答

<details>
<summary>AWS Outpostsとは？</summary>

- AWS が提供する“クラウドをそのままオンプレに持ち込む”仕組み。
  - オンプレでも AWS と同じ操作性で使える！
  - データをクラウドに出せない環境でも AWS が使える！
  - レイテンシが極めて低い！
- Outpost（アウトポスト） は英語で「前哨基地」「拠点」「遠隔地の基地」という意味を持つ言葉。

</details>

<details>
<summary>Change Data Capture（CDC） と CloudTrail はどちらも“イベントを記録する仕組み”ですが、記録対象が根本的に異なります。両者がそれぞれ何を記録する仕組みなのか説明してください。</summary>

- Change Data Capture は、データベース内部の 行レベルの変更（INSERT / UPDATE / DELETE） を記録する仕組み。
- CloudTrail は、AWS 上で実行された API 操作ログ（誰が何の操作をしたか） を記録する仕組み。

データそのものの変化と、操作そのものの記録という点が本質的に異なる。
</details>

<details>
<summary>AWS Data Exchangeとは？</summary>
データの “売買・取引” を行う場所である。外部のデータ（気象・位置・金融・ヘルスケアなど）を “AWS 上で安全に購読・配布できるマーケットプレイス”。
</details>

<details>
<summary>AWS Lake Formation とは？</summary>

- データレイク（S3 を中心とした全社データ基盤）を “簡単・安全・統一的に” 作るためのマネージドサービス。
- 実体としては Glue・IAM・S3 などをまとめて扱いやすくした “ラッピングレイヤー” に近く、
  - データレイク専用の 独自権限管理（Lake Formation Permissions） を持つのが特徴。
- IAM や Glue を個別に組み合わせて構築するよりも、
  - データレイク特化でアクセス制御・ガバナンスを簡単にする仕組みという位置づけ。
  
</details>

<details>
<summary>Amazon AppFlowとは？</summary>

- SaaS（Salesforce、Slack、Zendesk、Google Workspace など）と AWS（S3、Redshift、EventBridge など）をノーコードで双方向にデータ連携できる統合サービス。
- コードを書かずに、以下のデータ同期・変換・フィルタリングを定期実行できる “データ連携パイプライン” のマネージド版という位置づけ。
  - SaaS → S3
  - SaaS → Redshift
  - S3 → SaaS
  
</details>

<details>
<summary>AWS Batchとは？</summary>

  - AWS Batch は、コンテナを使ったバッチ処理をフルマネージドで実行するサービス。
- EC2 / Fargate を自動で起動・停止し、ジョブキューに投げたバッチ処理をスケーラブルに実行できる。
- 位置づけとしては「バッチ処理のための ECS/EKS のラッパ」 のようなイメージ

</details>

<details>
<summary>Amazon Lexとは？</summary>
- 音声・テキストによるチャットボットを構築するためのフルマネージドサービス。
- Alexa と同じ自然言語理解（NLU）・音声認識（ASR）技術がベースになっており、ユーザーの発話や入力テキストから意図（Intent）やスロットを抽出し、対話型アプリケーションを簡単に作れる。
- 主な用途はカスタマーサポートボット、社内問い合わせボット、音声インターフェイスなど。
</details>

<details>
<summary>Amazon Managed Grafanaとは？</summary>

<img width="1920" height="1081" alt="image" src="https://github.com/user-attachments/assets/54d6d501-9419-4514-96b8-2e58e521919f" />

- Amazon Managed Grafana は、Grafana を AWS がフルマネージドで提供する可視化・ダッシュボードサービス。
- Grafana サーバーの構築・運用・認証連携・スケーリングを AWS が肩代わりし、CloudWatch、Prometheus、OpenSearch、Amazon Timestream、Redshift などAWS データソースとの統合も簡単にできる。

</details>

<details>
<summary>AWS Directory Serviceとは？</summary>
  
- AWS Directory Service は、Active Directory（AD）を AWS 上で使えるようにするマネージドサービス群。
  - AWS Managed Microsoft AD は本物の Microsoft AD を AWS がフルマネージド提供
  - AD Connector はオンプレ AD に“中継接続”するだけの軽量版

</details>

<details>
<summary>Amazon Detectiveとは？</summary>
- Detectiveは「探偵」の意味
- Amazon Detective は、セキュリティインシデントの原因調査を支援する“可視化・分析サービス”。
- GuardDuty・CloudTrail・VPC Flow Logs などのログを自動で取り込み、ユーザー、IP、リソースごとの 行動グラフ（Graph） を生成して、“何が起きたのか”“どこから侵害が始まったのか” を追跡しやすくする。
</details>
